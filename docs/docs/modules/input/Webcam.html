Webcam

The webcam allows you to get a RenderBuffer of the real webcam on the device running Retro Gadgets. You can use this to draw the camera feed to a screen with a VideoChip.

Properties

RenderTarget - VideoChip

The VideoChip this camera is streaming contents to. Only this VideoChip can draw the RenderBuffer retrieved by this Webcam.

Methods

GetRenderBuffer() RenderBuffer

Gets the camera RenderBuffer. The render buffer obtained can then be fed to the DrawRenderBuffer method of the VideoChip module. Only the VideoChip defined as RenderTarget can draw the buffer.
